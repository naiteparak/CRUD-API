# Crud API

Simple Crud API by using vanilla node.js 

# Installation 
 
First of all you need to install all packages with command:

```bash
npm install
```

After installing all packages you need to create .env file from the .env.example file 


# Running 

App have a few methods of running 

- Development mode 
- Production mode 
- Multiple mode

For running app in **Development** mode you need to run this command:

```bash
npm run start:dev
```

For running app in **Production** mode you need to run this command:

```bash
npm run start:prod
```

For running app in **Multiple** mode you need to run this command:

```bash
npm run start:multi
```

# Tests

App also have tests for enpoints

For the test enpoints run this command: 

```
npm run test
```

# How to use?

After running app we can send some requests on endpoint **api/users**

- GET: get all users 
- GET/userId: get user by id
- POST: create new user
- PUT/userId: update user by id
- DELETE/userId: delete user by id
